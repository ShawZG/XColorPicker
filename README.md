# XColorPicker
pick color in windows screen

### compiler environment:
1. Qt 5.12.4
2. MinGW 64-bit
3. Win10 Qt Creator

You can visit my [blog](https://blog.csdn.net/shawzg/article/details/104041896) for more information.
